# Generate Component

Use this command when `$ARGUMENTS` is a component name and target folder.

## Instructions

1. Determine target path under `packages/excalidraw` or `excalidraw-app`.
2. Generate a functional component with typed props.
3. Add/update a colocated test with a basic render assertion.
4. Keep exports consistent with surrounding module style.
5. Return created/updated files and verification commands.

## Expected Output Format

- Created files
- Updated files
- Why this file shape
- Verification commands
- Usage snippet

## Example Input

`Button in packages/excalidraw/components`
